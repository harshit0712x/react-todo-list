# React To-Do List (Multi-Tab)

This is a single-page React application that allows users to manage
multiple to-do lists using a tab-based interface (similar to browser tabs).

## Features
- Create multiple to-do lists (tabs)
- Add tasks to each list
- Mark tasks as completed using checkboxes
- Data persists using browser localStorage
- No backend or database used

## Tech Stack
- React (Vite)
- JavaScript
- Git & GitHub

## Purpose
This project was built to learn GitHub workflows while implementing
a basic React application.
